## MODULE 4: ADVANCED ZAPIER HACKS (Pages 36-42)

### Hack #1: The Delay Trick

Most people don't know you can add delays between actions. This is powerful because it lets you create sequences.

**Example:**
```
Trigger: New lead
Action 1: Send email immediately
Action 2: Wait 1 day
Action 3: Send follow-up email
Action 4: Wait 3 days
Action 5: Send offer email
```

**Why it's powerful:** You can create multi-step sequences without needing multiple Zaps.

**How to use it:** Use the "Delay" action in Zapier to wait before the next action.

### Hack #2: The Conditional Logic Trick

You can add conditional logic to make automations smarter.

**Example:**
```
Trigger: New lead
Decision: Is lead from USA?
If YES → Send USA welcome email
If NO → Send International welcome email
```

**Why it's powerful:** You can personalize automations based on data.

**How to use it:** Use the "Filter" or "Conditional" action in Zapier.

### Hack #3: The Data Transformation Trick

You can transform data before sending it to the next app.

**Example:**
```
Trigger: New form submission with name "John Smith"
Transform: Split name into first name and last name
Action: Create CRM contact with first name and last name
```

**Why it's powerful:** You can clean up messy data automatically.

**How to use it:** Use the "Formatter" action in Zapier.

### Hack #4: The Multi-Step Workflow Trick

You can create complex workflows by chaining multiple Zaps together using webhooks.

**Example:**
```
Zap 1: Lead form → Webhook
Zap 2: Webhook → CRM
Zap 3: Webhook → Email
Zap 4: Webhook → Slack
```

**Why it's powerful:** You can create enterprise-level automations.

**How to use it:** Use webhooks to connect multiple Zaps.

### Hack #5: The Error Handling Trick

You can handle errors gracefully so automations don't break.

**Example:**
```
Trigger: New lead
Try: Create CRM contact
If Error: Send alert email to you
```

**Why it's powerful:** You'll know immediately if something breaks.

**How to use it:** Use the "Catch" action in Zapier.
